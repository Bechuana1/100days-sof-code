This tutorial is about objects and there methods

1. Object.Keys()

this method returns an arry of objects properties

        const person = { 
            name: 'John',
            age: 30,
            gender: 'male'
          };
          
          const keys = Object.keys(person);
          console.log(keys);  // Output: ['name', 'age', 'gender']

2. Object.assign(target, source)

this method concatenates two objects and returns as one.

        const target = {
            a : 1,
            b:2
        }
        
        const source ={
            d:3,
            e : 4
        }
        var full = Object.assign(target,source);
        console.log(full);

        output : {a: 1, b: 2, d: 3, e: 4}

3. Object.Entries()

this method returns an array of KEY : VALUE pairs


            const obj1 = {
                name : 'Michael bechuana',
                age : 21
                school : 'jkuat',
                joined : 2021
            }

            var entries =   Object.entries(obj1);
            console.log(entries);
output : 0 : (2) ['name', 'Michael bechuana'] 1 : (2) ['age', 21] 2 : (2) ['school', 'jkuat'] 3 : (2) ['joined', 2021] length : 4

This method make the object unaable to be altered, new properties added or removed
