Array methods

1. Array.lenght()

This returns tht length of the Array. Always remember counting starts at [0] for Arrays

            let fruits = ['mangoes','kiwi' , 'passion'];
            console.log(fruits.length);

            //output : 3

2. Array.push()

Add element to the end of Array and takes O(1)

           
            fruits.push('citrus');
            console.log(fruits);

            //output : ['mangoes', 'kiwi', 'passion', 'citrus']

3. Array.pop()

remove the last element from the array and takes O(1)

            let fruits = ['mangoes','kiwi' , 'passion'];
            fruits.pop();


            //output : ['mangoes', 'kiwi']

4. Array.toString()

This method converts an Array to string separated by commas

            let fruits = ['mangoes','kiwi' , 'passion'];
            console.log(fruits.toString());

        //output : mangoes,kiwi,passion
